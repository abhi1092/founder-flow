# Example Output

These files are from a real FounderFlow validation run on the idea: **"AI receptionist for independent dental clinics that handles missed calls, appointment reminders, and review requests"**

## Agent Outputs (the best part)

The three specialist agents ran in parallel and produced detailed, research-grounded analysis:

- **[sample-idea-validation.json](sample-idea-validation.json)** — Problem severity ($100K+ annual loss from missed calls), target customer clarity (96K independent practices), value proposition strength, 6 risk factors, 6 testable assumptions
- **[sample-competitor-analysis.json](sample-competitor-analysis.json)** — Direct competitors (Arini, Weave, Adit, etc.), indirect competitors, substitutes, current workarounds, pricing patterns, positioning gaps
- **[sample-customer-discovery.json](sample-customer-discovery.json)** — Interview targets, Mom Test-style discovery questions, demand signals, weak-signal warnings, first validation experiment design

## Integrator Output

- **[sample-directive.json](sample-directive.json)** — The Evidence Integrator's ResearchDirective identifying gaps and routing follow-up questions to specific agents
- **[sample-synthesis.json](sample-synthesis.json)** — Final ValidationThesis with verdict

## Brief

- **[sample-brief.html](sample-brief.html)** — The self-contained HTML Startup Validation Research Brief (open in browser)
- **[sample-brief.json](sample-brief.json)** — Machine-parseable JSON version
